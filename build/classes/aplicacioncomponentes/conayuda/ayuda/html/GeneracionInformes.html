Ayuda - Generación de informes

Cómo generar un informe

La generación de informes es el proceso de recopilación, organización y presentación de datos de manera estructurada y legible para su análisis, toma de decisiones o presentación a usuarios finales

  • Vamos al menu "Informes -> Productos"
  • Buscamos/filtramos mediante las lista la clase de productos que queremos. Si hay resultados, se habilitarán los botones de generar informes
  • Le damos al botón buscar para realizar para filtrar

  • Para generar un informe tenemos 3 opciones:

    1. Informes normales > Genera un informe estándar con una imagen de los resultados de la busqueda de productos
    2. Informes y gráfico Genera un informe que incluye gráficas junto con los datos de productos.
    3. Informes con agrupación de datos Genera un informe que agrupa los datos de productos según ciertos criterios,como por categoría
  • Con el botón "Reinciar", se limpian los datos del formulario"

Búsqueda de Producto

La búsqueda de productos se efectuará al seleccionar un elemento de las listas y darle al botón buscar.
Permite encontrar productos específicos en la tabla utilizando criterios de búsqueda:

  • Búsqueda por País: Busca productos por el país del proveedor.
  • Búsqueda por Categoría: Busca productos por su categoría
  • Búsqueda por Proveedor Busca productos por su proveedor
